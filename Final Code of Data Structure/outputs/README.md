# Data Structure Final Code

## 📌 Project Description
This project contains the final implementation of various Data Structures in C++. It demonstrates the working and operations of Linked Lists, Stack, and Queue.

## 🚀 Implemented Data Structures

### 🔹 Linked Lists
- Singly Linked List
- Singly Circular Linked List
- Doubly Linear Linked List
- Doubly Circular Linked List

### 🔹 Stack
- Stack using Linked List
- Push operation
- Pop operation
- Display elements

### 🔹 Queue
- Queue using Linked List
- Enqueue operation
- Dequeue operation
- Display elements

## ⚙️ Features
- Insertion (Beginning, End, Position)
- Deletion (Beginning, End, Position)
- Push & Pop (Stack)
- Enqueue & Dequeue (Queue)
- Display elements
- Count number of nodes
